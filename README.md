# Tensei Shitara

Tensei Shitara is a responsive e-commerce anime store that provides anime fans with a visually appealing and easy-to-use online shopping platform. It is built using modern frontend and backend technologies to ensure scalability, performance, and long-term maintainability for real-world deployment.

## Installation & Setup
1. Clone Repository
```git clone https://github.com/elmesiashu/tenseishitara.git```
2. Backend Setup
```cd backend```
```npm install```
```npm run dev```
3. Create a .env file in backend
```PORT=5000```
```DATABASE_URL=your_database_url```
```JWT_SECRET=your_secret_key```
4. Frontend Setup
```cd frontend```
```npm install```
```npm run dev```

## Usage
Open browser at:
```http://localhost:3000```

## User Interface Design
The website follows a modern e-commerce layout that includes:
- Header with logo, search bar, and action icons
- Promotional banner with call-to-action button
- Popular products carousel
- Product cards with images, prices, and buttons
- Clean color scheme with strong contrast
The design focuses on simplicity, accessibility, and usability.

## Deployment
1. Build frontend
2. Configure environment variables
3. Deploy backend (Render, Railway, AWS, etc) 
4. Host frontend (Vercel)
